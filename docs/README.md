## Files

| File | Purpose |
|------|---------|
| `Mulema-Full-Testing-Guide.html` | Complete testing guide (printable) |
| `Mulema-UAT-Signoff.html` | UAT sign-off sheet with Pass / Fail / N/A columns |

## Save as PDF

1. Open the `.html` file in **Chrome** or **Safari**.
2. **File → Print** (or `Cmd+P` / `Ctrl+P`).
3. Destination: **Save as PDF**.
4. Paper: **A4**, Margins: **Default**.
5. Save.

## Recommended workflow

1. **Tester** — Follow `Mulema-Full-Testing-Guide.html` step by step.
2. **QA lead** — Record results on `Mulema-UAT-Signoff.html` (mark P/F/N/A, note defects).
3. **Sign-off** — Complete the signature block at the end of the UAT sheet.
4. **Critical items (★)** on the sign-off sheet must pass before production approval.
